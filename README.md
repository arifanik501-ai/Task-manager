# Task-manager

This repository hosts two browser-based projects:

## TaskFlow
A polished task manager with a calendar, themes and cloud-sync UI.
Open [`index.html`](./index.html) in a browser, or visit the GitHub Pages site.

## ষড়যন্ত্র (Shorojontro) — Political Strategy Game
A Bengali-language political strategy game (mobile-first, HTML/CSS/JS).
Open [`shorojontro/index.html`](./shorojontro/index.html) in a browser, or visit the GitHub Pages path
`/shorojontro/`.

Key features:
- 5টি চরিত্র (নেতা, ব্যবসায়ী, সাংবাদিক, জেনারেল, বিচারক)
- ৫ ফেজের রাউন্ড: ইভেন্ট → অ্যাকশন → আলোচনা → ভোটিং → এন্ড রাউন্ড
- কার্ড সিস্টেম (ষড়যন্ত্র + পাওয়ার), মন্ত্রণালয় দখল, জোট, দুর্নীতি ও স্ক্যান্ডাল
- ৩টি গেম মোড: ক্যাজুয়াল / স্ট্যান্ডার্ড / হার্ডকোর
- লিডারবোর্ড স্থানীয়ভাবে সংরক্ষিত (localStorage)
