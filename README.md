# হিসাব

হিসাব is a mobile-first monthly expense tracker built with HTML, CSS, and vanilla JavaScript.

## Features

- Splash and first-time monthly budget setup
- Monthly budget, spent, remaining balance, and usage progress
- Add, edit, delete, and undo expenses
- Category icons, descriptions, dates, and times
- Today dashboard, monthly history, search, filters, and sorting
- Category breakdown, donut chart, daily bar chart, cumulative trend, and top expenses
- Built-in calculator with “Add as Expense”
- Settings for budget, currency, dark mode, reminders, reset, CSV export, and print/PDF export
- LocalStorage-only privacy model
- PWA manifest and service worker for installable/offline use

## Run locally

Open `index.html` in a browser, or serve the folder with any static server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
